# CNN
## Công nghệ sử dụng
Python, NumPy, Matplotlib, PyTorch, Scikit-learn.
## Cách hoạt động
- Chuẩn bị và xử lý dữ liệu ảnh  
- Dùng CNN để học đặc điểm từ ảnh  
- Huấn luyện mô hình để giảm sai số  
- Dự đoán và so sánh với kết quả thật  
- Thử thay đổi các tham số để xem ảnh hưởng  
## Kết quả
- Mô hình có thể nhận diện ảnh khá tốt  
- Tăng số lớp hoặc số neuron giúp học tốt hơn nhưng có thể chậm hơn  
- Học nhiều lần giúp giảm lỗi nhưng dễ bị sai với dữ liệu mới  
- Thêm dropout giúp mô hình ổn định hơn  
## Kết luận
CNN phù hợp cho bài toán ảnh, nhưng cần chọn tham số phù hợp để đạt kết quả tốt.
